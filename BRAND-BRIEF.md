# Block 90' — Claude Design Master Brand Brief
**Paste this entire document at the start of every Claude Design session.**
From this point, every asset you generate must conform to this brand system exactly.

---

## 1. Brand Identity

| Property | Value |
|---|---|
| Brand name | Block 90' |
| Tagline | Football. No filter. |
| Channel type | TikTok football (soccer) channel |
| Tone | Street, raw, urban, high-energy, opinionated |
| Audience | Football fans 16–35, TikTok-native |
| Primary format | 9:16 vertical (1080×1920px) |

---

## 2. Colour System

Use these exact hex values. No substitutions.

| Token | Hex | Role |
|---|---|---|
| Bloc Red | `#D62828` | Primary accent — match reactions, breaking news, top bars |
| Hot Red | `#FF3333` | Live moments, hot takes, left-bar accent |
| Pitch Gold | `#F4A01C` | Transfers, eyebrows, category labels, fee values |
| Pitch Green | `#1D9E75` | Tactical breakdowns, pitch backgrounds |
| Data Blue | `#378ADD` | Stats, data posts, bar charts |
| Bronze | `#8B6914` | Football culture, throwback, history posts |
| Terrace White | `#F0EDE8` | All headlines and body text |
| Block Black | `#141414` | Card and surface backgrounds |
| Deep Black | `#0A0A0A` | Canvas background — permanent, no exceptions |
| Charcoal | `#1E1E1E` | Borders, dividers |
| Slate | `#5A5A5A` | Secondary text, muted labels |
| Deep Slate | `#3A3A3A` | Tertiary text, source tags |

**Rules:**
- Background is always `#0A0A0A`. Never white, cream, grey, or stadium green.
- Maximum 3 colours per post.
- Gold (`#F4A01C`) is for eyebrows and labels only — never body text.
- No gradients of any kind.
- No drop shadows or glows.

---

## 3. Typography

| Role | Font | Weight | Case | Size guide |
|---|---|---|---|---|
| Display / Wordmark | Bebas Neue | Regular | Uppercase | 48px+ for heroes, 15px for footer wordmark |
| Headline | Barlow Condensed | 700–800 | Uppercase | 18–24px on posts |
| Eyebrow / Label | Barlow Condensed | 600 | Uppercase | 8–10px, letter-spacing 3–4px |
| Body / Caption | Barlow | 400 | Sentence case | 11–13px |
| Score / Data | Bebas Neue | Regular | — | 36–64px |

**Rules:**
- Bebas Neue is used ONLY for: wordmark, scores, large hero numbers, quote marks.
- Barlow Condensed is the workhorse for all headlines on posts.
- Never mix fonts outside this system.
- Eyebrow text always has letter-spacing of at least 3px.

---

## 4. Logo & Wordmark

**Wordmark:** `BLOCK 90'` — "BLOCK" in `#F0EDE8`, "90'" in `#D62828`. Bebas Neue, letter-spacing 2–3px.

**Icon mark:** Pentagon shape (square with bottom-right corner cut at ~72% height). Red fill `#D62828`. Text inside: "B90" in Bebas Neue, `#F0EDE8`.
- Clip-path: `polygon(0 0, 100% 0, 100% 72%, 72% 100%, 0 100%)`
- Never use a circle or square crop for the icon.

**Watermark (post footer):** Wordmark at 18% opacity — `rgba(240,237,232,0.18)` for "BLOCK", `rgba(214,40,40,0.35)` for "90'". Always bottom-left of post.

## 5. Content Categories & Visual Signals

Each content type has a unique colour stripe and layout signal. Apply these exactly.

### Match Reaction
- **Stripe:** 3–4px top bar, `#D62828`
- **Eyebrow:** Competition + matchday, `#F4A01C`, Barlow Condensed 600, 9px, letter-spacing 3px
- **Score:** Bebas Neue, large, `#F0EDE8`; dash between scores in `#D62828`
- **Reaction headline:** Barlow Condensed 700, uppercase, `#F0EDE8`; one key word in `#D62828`
- **Ghost element:** "90" in Bebas Neue, `rgba(214,40,40,0.06)`, huge, bottom-right corner
- **Status badge:** "Full Time" / "Live" / "Half Time" — Live uses animated dot `#FF3333`

### Hot Take
- **Stripe:** 3px left vertical bar, `#FF3333`, full height
- **Badge:** "HOT TAKE" pill, `#D62828` fill, `#F0EDE8` text, Barlow Condensed 600, 8px
- **Quote mark:** Bebas Neue, 64px, `#D62828`, line-height 0.75
- **Statement:** Barlow Condensed 700, uppercase, `#F0EDE8`; highlight word in `#F4A01C`
- **CTA:** Bottom-right, Barlow Condensed 600, `#5A5A5A` — e.g. "Agree or nah? ↓"

### Transfer News
- **Stripe:** 3px top bar, `#F4A01C` (gold — NOT red)
- **Eyebrow:** "TRANSFER UPDATE", Barlow Condensed 600, 8px, `#F4A01C`
- **Club flow:** FROM club in `#5A5A5A` → arrow `#D62828` → TO club in `#F0EDE8`
- **Player name:** First name in Bebas Neue `#F0EDE8`, last name larger in `#F4A01C`
- **Fee card:** Dark inset `#141414`, border `#2E2E2E`, "REPORTED FEE" label in `#5A5A5A`, value in `#F4A01C` Bebas Neue
- **Status pill:** Rumour = grey outline / Breaking = `#D62828` fill / Official = `#F4A01C` fill black text
- **Source:** Bottom-right, `#3A3A3A`

### Tactical Breakdown
- **Stripe:** 3px left vertical bar, `#1D9E75`
- **Eyebrow:** "TACTICAL BREAKDOWN", `#F4A01C`
- **Headline:** Barlow Condensed 800, uppercase; formation number in `#D62828`
- **Pitch diagram:** Background `#0D1A0F`, border `#1A3020`; our team = `#D62828` filled dots with position labels; opposition = `#5A5A5A` outline dots
- **Key points:** Number in `#D62828` Bebas Neue; text in `#C0BBBA` Barlow 400, 11px

### Stats & Data
- **Background:** `#04193A` (very dark blue — only category that breaks the pure black rule)
- **Stripe:** 3px left bar, `#378ADD`
- **Headline:** Barlow Condensed 800, `#F0EDE8`
- **Bars:** `#378ADD` fill, `#0C2E5A` track; rank number in `#378ADD`
- **#1 entry:** Slightly wider bar, name in `#F0EDE8` vs `#5A5A5A` for others

### Football Culture / Throwback
- **Stripe:** 3px left bar, `#8B6914` (bronze)
- **Eyebrow:** "FOOTBALL CULTURE", `#8B6914`
- **Highlight word:** `#8B6914` — never red or gold for this category
- **Year/era:** Bebas Neue large, `#8B6914` at low opacity, background element
- **Body:** Barlow 400, 12px, `#C0BBBA`
- **Footer tag:** "#Throwback", `#3A3A3A`

## 6. Photo Background Posts

When using a photo as the background (all post types support this):

| Property | Value |
|---|---|
| Photo layer | Full bleed, `background-size: cover`, `background-position: center` (adjust to subject) |
| Overlay type | Gradient — strong at bottom (`rgba(10,10,10,0.95)`), fades to light at top (`rgba(10,10,10,0.15)`) |
| Overlay darkness | 50–70% for clean portraits / 70–90% for busy stadium/action shots |
| Accent bar | Always present on top (or left for hot takes) — same colour rules as text-only posts |
| Text contrast | All text remains `#F0EDE8` — the overlay ensures readability on any photo |
| Watermark | Wordmark at 18% opacity, bottom-left, always visible above overlay |

**Overlay gradient (default):**
```
linear-gradient(to bottom,
  rgba(10,10,10,0.15) 0%,
  rgba(10,10,10,0.35) 40%,
  rgba(10,10,10,0.75) 65%,
  rgba(10,10,10,0.95) 100%
)
```

**Rules:**
- Never remove the overlay — text must always be readable
- Never use a coloured overlay — dark only, always `rgba(10,10,10,x)`
- Photo focus point: face/subject at top-center for player shots; crowd/stadium at center for atmosphere
- Accent bar colour follows the same category rules (red for match, gold for transfer, etc.)

## 7. Layout Rules (All Posts)

- **Format:** 9:16 vertical, 1080×1920px
- **Padding:** 16px all sides minimum
- **Divider lines:** `0.5px solid #2E2E2E` — thin only, never thick
- **Card corners:** No border-radius — sharp corners on all post cards
- **Accent bars:** Always 3–4px, flat colour, no gradients
- **Footer:** Always present — wordmark left, category tag or status right
- **No rounded corners on posts**
- **No drop shadows**
- **No gradients**
- **No glow effects**

---

## 7. Voice & Copy Rules

- Lead with the feeling, not the fact
- Punchy — max 20 words on a headline
- All-caps for headlines via Barlow Condensed
- Gold for categories, red for urgency — never swap
- CTAs drive comments: "Agree or nah? ↓" / "Change my mind ↓" / "Rate this ↓"
- Never say "we" — the brand speaks directly: "Arsenal are different this season."

---

## 8. What NOT to Do

- ❌ White or light backgrounds
- ❌ More than 3 colours in one post
- ❌ Rounded corners on post cards
- ❌ Drop shadows or glows
- ❌ Gold text in body copy
- ❌ Bebas Neue for body text
- ❌ Gradients of any kind
- ❌ Mixing fonts outside the 3-font system
- ❌ Red accent bar on transfer posts (gold only)
- ❌ Green background outside of tactical posts
